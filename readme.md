## Installation Steps

### setup python env 
python version : 3.11.1

### os : windows
### install virtualenv 
```
pip install virtualenv
```
### create virtualenv 
```
python -m venv venv
```
### to active virtualenv 
```
.\venv\Scripts\activate
```
### install requirement packages
```
pip install -r requirements.txt
```

### run project 

```
python manage.py runserver
```

### Browse

```
http://127.0.0.1:8000/
```
